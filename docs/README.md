## 🏗️ Architecture Overview

```
Open Design Platform
├── Frontend (Next.js 14)
│   ├── Visual Canvas Editor
│   ├── Monaco Code Editor
│   ├── AI Assistant
│   ├── Real-time Collaboration
│   └── Project Management
├── Backend (Node.js + TypeScript)
│   ├── REST API
│   ├── WebSocket Server
│   ├── Authentication
│   ├── AI Job Queue
│   └── Asset Management
└── Infrastructure
    ├── MongoDB Database
    ├── Redis Cache
    ├── AWS S3 Storage
    └── Docker Containers
```
